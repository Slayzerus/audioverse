Rysunek 4.4 — Diagram przepływu autoryzacji JWT i OAuth

CO PRZEDSTAWIA:
Diagram przepływu autentykacji i autoryzacji w AudioVerse:
Login → Identity Server → JWT token → SignalR z Bearer token →
OAuth flow (Google/Spotify/Twitch/Discord) → callback → token refresh.
Widoczne role: User, Admin i ich uprawnienia do endpointów.
