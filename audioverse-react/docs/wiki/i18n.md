# Internacjonalizacja (i18n)

## Biblioteka
`react-i18next` + `i18next` z backendem HTTP.

## Obsługiwane języki
- **pl** — Polski (domyślny)
- **en** — Angielski

## Użycie w komponentach

```typescript
import { useTranslation } from 'react-i18next';

const Component = () => {
    const { t } = useTranslation();
    return <h1>{t('page.title')}</h1>;
};
```

### Interpolacja
```typescript
t('karaokeManager.playerN', { id: playerId })
// → "Player 3"
```

### Wartości domyślne
```typescript
t('editor.songMetadata', 'Song Metadata')
// → Fallback do "Song Metadata" jeśli klucz nie istnieje
```

## Pliki tłumaczeń

Zlokalizowane w `public/locales/{lang}/translation.json`:

```
public/
  locales/
    pl/
      translation.json    # Tłumaczenia polskie
    en/
      translation.json    # Tłumaczenia angielskie
```

## Zmiana języka

```typescript
import { useTranslation } from 'react-i18next';

const LanguageSwitcher = () => {
    const { i18n } = useTranslation();
    return (
        <select onChange={e => i18n.changeLanguage(e.target.value)}>
            <option value="pl">🇵🇱 Polski</option>
            <option value="en">🇬🇧 English</option>
        </select>
    );
};
```
